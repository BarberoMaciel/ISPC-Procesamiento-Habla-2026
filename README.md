# Técnicas del Procesamiento del Habla

## Descripción

Este repositorio corresponde a la materia **Técnicas del Procesamiento del Habla**.

En el marco de esta materia, espero aprender a procesar datos en formato de audio de manera eficiente, así como incorporar herramientas de procesamiento y modelado orientadas al desarrollo de soluciones basadas en datos.

## Áreas de interés en Ciencia de Datos

Mis principales intereses dentro de la Ciencia de Datos incluyen:

* Desarrollo de soluciones basadas en Inteligencia Artificial
* Procesamiento de datos
* Modelado de datos y aplicación de modelos predictivos
